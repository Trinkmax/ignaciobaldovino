Resumen – SellPilot AI

SellPilot AI fue un proyecto avanzado de automatización de prospección y conversaciones en Instagram, desarrollado íntegramente con código, antes del boom de plataformas low-code como n8n aplicadas a automatización conversacional.

El sistema estaba diseñado para detectar, analizar y contactar prospectos de forma inteligente, y además responder conversaciones automáticamente, guiando al usuario según su perfil, mensajes y nivel de interés.

🧠 Inteligencia y personalización

SellPilot incorporaba un sistema RAG (Retrieval Augmented Generation) basado en:
	•	Embeddings
	•	Base de datos vectorial
	•	Contexto histórico de conversaciones
	•	Información scrapeada del perfil del prospecto

Antes de enviar un mensaje, el sistema:
	•	Scrapeaba el perfil de Instagram (bio, nombre, keywords, tipo de negocio).
	•	Analizaba el perfil del posible cliente.
	•	Generaba mensajes personalizados según el rubro, lenguaje y características del prospecto.
	•	Ajustaba la respuesta según el estado de la conversación.

Esto permitía mensajes mucho más humanos y relevantes que el spam tradicional.

🤖 Automatización conversacional
	•	Automatizaba el primer mensaje.
	•	Respondía mensajes entrantes automáticamente.
	•	Usaba lógica por estados (nuevo lead, interesado, seguimiento, derivación).
	•	Permitía derivar la conversación a un humano en el momento adecuado.

⚙️ Stack técnico

Todo el sistema estaba construido 100% con código, sin herramientas visuales:
	•	Python
	•	Selenium y Playwright para scraping y automatización web
	•	Simulación de comportamiento humano (pausas, scroll, variaciones)
	•	Bases de datos para historial y control de usuarios
	•	Sistema de métricas y dashboard

📌 Estado actual del proyecto

SellPilot AI es hoy un proyecto finalizado.
No porque haya fallado, sino porque el ecosistema evolucionó:
actualmente existen herramientas más rápidas y flexibles (como n8n + IA) que reducen drásticamente el tiempo de implementación de este tipo de soluciones.

En su momento, SellPilot funcionaba muy bien y estaba adelantado a su época, resolviendo problemas que hoy recién se popularizaron.
