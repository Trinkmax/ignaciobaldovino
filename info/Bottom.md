Bot de reservas y sistema de gestión para Bottom Bar

El proyecto consistía en crear un sistema integral de reservas para el bar boliche Bottom Bar, combinando automatización por WhatsApp con un panel web de gestión, pensado específicamente para el funcionamiento nocturno de un bar/boliche.

🤖 Bot de WhatsApp

El bot se encargaba de toda la interacción con los clientes que querían reservar:
	•	Respondía mensajes automáticamente, las 24 horas.
	•	Guiaba al usuario paso a paso para tomar la reserva.
	•	Solicitaba datos clave: nombre, cantidad de personas, fecha y horario.
	•	Validaba la disponibilidad según la capacidad real del local.
	•	Confirmaba la reserva o informaba si no había cupo disponible.
	•	Registraba cada reserva en una base de datos centralizada.

El objetivo principal era evitar errores humanos, demoras en la respuesta y sobre-reservas, algo muy común en noches de alta demanda.

🖥️ Sistema web de gestión

El sistema web estaba pensado para el dueño o el staff de Bottom Bar, no para el público:
	•	Visualización en tiempo real de todas las reservas.
	•	Filtros por fecha, horario y estado.
	•	Control claro de cupos ocupados vs disponibles.
	•	Historial de reservas para control operativo.
	•	Información ordenada para la organización del ingreso al boliche.

🔗 Funcionamiento conjunto

El bot y el sistema web estaban totalmente conectados:
	•	El bot cargaba las reservas automáticamente.
	•	El panel web reflejaba esos datos en tiempo real.
	•	Si un horario se llenaba, el bot lo detectaba y dejaba de aceptar reservas para ese rango.

🎯 Valor para Bottom Bar
	•	Menos personal atendiendo WhatsApp.
	•	Respuestas rápidas y consistentes.
	•	Reservas ordenadas y centralizadas.
	•	Mejor experiencia para el cliente.
	•	Base sólida para escalar a eventos, listas VIP o sucursales.s
